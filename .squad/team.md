## Project Context

A sophisticated agentic system that bridges the GitHub Copilot SDK with the Microsoft Agent Framework to automate local repository maintenance. C# agents autonomously analyze branch stale dates, identify merge conflicts, and execute self-healing workflows. The system operates with pure local intelligence, providing developers with a self-governing workspace that handles tedious cleanup tasks while maintaining strict safety guardrails.

**Key Goals:**
- Intent-driven agentic orchestration using Copilot SDK
- Autonomous branch analysis and stale date detection
- Safe conflict resolution without human intervention
- Semantic code understanding for merge safety
- Comprehensive logging and safety guardrails
- Technical narrative & developer documentation
